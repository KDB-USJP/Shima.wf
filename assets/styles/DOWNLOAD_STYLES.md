# Style Thumbnail Packs

The style thumbnail images are not included in this repository due to their size.

## Download

Download the style packs from the link below and extract them into this directory:

  https://bit.ly/shima-v2

## Installation

1. Visit the link above to access the Shima public Google Drive folder.
2. Download the style pack folder(s) you want.
3. Place the downloaded folder(s) inside this directory:

       ComfyUI/custom_nodes/Shima/assets/styles/

   The final structure should look like:

       assets/
         styles/
           Still Life/
             A0.png
             A1.png
             ...
           Walking Woman/
             A0.png
             A1.png
             ...

4. Restart ComfyUI. The Shima Styler nodes will automatically detect the installed packs.
